# Beach Apartment Booking System

Sistema de reservas para aluguel de apartamento na praia desenvolvido em PHP e MySQL.

## Objetivo

Permitir que clientes visualizem informações do apartamento, consultem disponibilidade e realizem solicitações de reserva.

Além disso, fornecer uma área administrativa para gerenciamento de reservas, despesas e informações do imóvel.

## Tecnologias

* PHP
* MySQL
* PDO
* HTML5
* CSS3
* JavaScript (futuramente)

## Funcionalidades Planejadas

### Área Pública

* Página inicial
* Galeria de fotos
* Disponibilidade de datas
* Regras do imóvel
* Contato

### Área do Cliente

* Cadastro
* Login
* Solicitação de reserva
* Consulta de reservas

### Área Administrativa

* Gerenciamento de reservas
* Controle de despesas
* Gerenciamento de imagens
* Relatórios financeiros
